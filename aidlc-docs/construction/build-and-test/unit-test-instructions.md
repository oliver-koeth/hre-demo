# Unit Test Execution

## Run Unit Tests

### 1. Execute All Unit Test Projects
```bash
dotnet test tests/AuthModule.Foundation.Tests/AuthModule.Foundation.Tests.csproj --nologo
dotnet test tests/AuthModule.CoreSecurity.Tests/AuthModule.CoreSecurity.Tests.csproj --nologo
dotnet test tests/AuthModule.Governance.Tests/AuthModule.Governance.Tests.csproj --nologo
dotnet test tests/AuthModule.Integration.Tests/AuthModule.Integration.Tests.csproj --nologo
```

### 2. Review Test Results
- **Expected**: all test projects pass with 0 failures
- **Coverage**: collected via `coverlet.collector` when enabled in test runner/reporting pipeline
- **Reports Location**: `tests/**/TestResults/` (when generated by runner)

### 3. Fix Failing Tests
1. inspect failing output from `dotnet test`
2. fix implementation or test assumptions
3. rerun the failing project first, then rerun all projects
